# Attendance Tracking System
*Collaborative Major Subject Project | Completed during 3rd Year, 1st Semester*

This is a web-based attendance tracking application developed as part of a multi-semester collaboration under the guidance of Prof. Lucman Abdulrachman. The system provides teachers with an efficient digital platform to monitor student attendance, manage classroom records, and view attendance statistics in real time.

## Project Structure & Architecture
- **Frontend Development:** Built using the Quasar Framework and Vue.js to create a modern and responsive user interface.
- **Type Safety:** Implemented with TypeScript for cleaner code structure and reliable component development.
- **Backend & Database:** Integrated with Firebase for authentication, cloud data storage, and real-time attendance tracking.

## Leadership & Contributions
- **Project Team Leader:** Led the development team by assigning tasks, monitoring progress, reviewing outputs, and ensuring collaboration throughout the project lifecycle.
- **Component Development:** Collaboratively developed interactive tab panels and interface views.
- **Data Visualization Logic:** Assisted in structuring attendance tracking logic, student averages, and summary reports.
- **System Integration:** Organized scripts and integrated local components within the development environment using VS Code.
